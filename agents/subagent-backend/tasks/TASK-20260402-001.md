- 已在 getFormData 加了 v() 辅助函数处理 null
- 已在 update_daily_task 加了 status/done 同步逻辑

3. 完整测试：修改日期 -> 保存 -> 刷新页面 -> 确认日期已修改
【输入输出约束】
