
## 飞书文件上传权限修复

**问题**：飞书机器人无法发送Excel附件，只显示为文本路径

**原因**：
1. 飞书机器人缺少 `drive:file:upload` 等drive权限
2. 发送文件时用 `file_type=stream` 而不是其他类型

**解决**：
1. 在飞书开放平台给机器人开通权限：
   - `drive:drive`
   - `drive:file`
   - `drive:file:upload`

2. 上传文件用 `file_type=stream`
   ```
   POST https://open.feishu.cn/open-apis/im/v1/files
   ```

3. 发送文件消息用 `file_key`（不是 `file_token`）
   ```
   POST https://open.feishu.cn/open-apis/im/v1/messages
   msg_type: file
   content: {"file_key":"...", "file_name":"xxx.xlsx"}
   ```

**备注**：文件名含中文时API可能只识别为文本，改用ASCII文件名或先上传到drive再发消息更可靠。
